# Mineversion
A lightweight tool for server-side Minecraft mod versioning, written in Rust.

This package is an extremely early work in progress, but over time, it will support
- Server broadcasting of required mods
- Extremely lightweight (text-file) sharing of mod packs
- Integration with Prism Launcher, and later, CurseForge

This project will remain permanently open-source, and it is GPL-3.0 licensed.

Reach out with issues!

## Getting Started

- Install just via [your favorite package manager](https://just.systems/man/en/packages.html)
- In a terminal, `just setup`
